Question: What is a key property that must hold true for every node in a doubly linked list?

Possible Answers:
a) The next pointer of a node must point to the node that it started at.
b) The prev pointer of a node must point to the node that it started at.
c) The next pointer of a node must point to the node on the right of it.
d) The prev pointer of a node must point to the node on the left of it.

Key: b
