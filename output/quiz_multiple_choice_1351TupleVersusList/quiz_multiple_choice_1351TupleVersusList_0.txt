Question: Which of the following is a key difference between tuples and lists in Python?

Possible Answers:
a) Tuples are ordered, while lists are not.
b) Tuples are mutable, while lists are immutable.
c) Tuples can contain elements of different data types, while lists can only contain elements of the same data type.
d) Tuples are more efficient to iterate through than lists.

Key: b
